main - главный файл, который запускает меню и перенаправляет в остальные файлы при выборе соответсвующей позиции
Classes - файл для хранения классов
Functions - файл для хранения всех вспомагательных функций
в папке images лежат используемые изоображения
в папке music лежит используемая музыка
Scores.txt - файл, в который записываются результаты прохождения
для запуска игры запустить файл main.py
